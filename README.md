# midi-stream
Stream midi over the internet.

## Contributing

See [`CONTRIBUTING.md`](CONTRIBUTING.md) for details.

## Install dependencies
```sh
yarn
```

## Run the server
```sh
yarn start
```

## Connect to the server
```sh
yarn start [server ip address]
```
## Help
```sh
yarn start --help
```

Uses port 25144, which was chosen randomly.
